9 · Accessibility & coherence rules

  1. Ink clears 4.5:1 on every surface tier it can sit on; team ink is contrast-checked programmatically before use.
  2. Every interactive element shows a visible 2px accent focus outline (tab through this page).
  3. Numbers right-align with tabular-nums; the queried metric column gets .is-metric.
  4. Sort headers expose aria-sort; tabs/segments use aria-selected/aria-pressed; popovers live at --z-pop, in-flow, no portals.
  5. Accent is earned: solid accent for primary actions and selection only; accent text always uses --accent-ink.
  6. Team theming scopes to the component owning the team context — never the page shell; neutrals never recolor.
  7. All motion (.yb-enter, shimmer, draw-ins) collapses under prefers-reduced-motion.
